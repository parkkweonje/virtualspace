# 🔮 사주꿈 (SajuKkum)

사주팔자 · 오늘의 운세 · 꿈해몽을 한 곳에서 볼 수 있는 **정적(static) 웹사이트**입니다.
생년월일시만 입력하면 만세력 기반으로 사주팔자와 오행 분포를 분석해 드립니다.
서버가 필요 없어 **무료 GitHub Pages**로 운영됩니다.

- 배포 도메인: <https://www.sajukkum.com>

## 📄 구성

| 파일 | 내용 |
|------|------|
| `index.html` | 메인 페이지 — 사주팔자 · 오늘의 운세 · 꿈해몽 탭 |
| `saju.js` | 사주 계산 엔진(년/월/일/시주·오행) + 운세·꿈해몽 데이터 |
| `style.css` | 신비로운 밤하늘 테마 스타일 (반응형) |
| `CNAME` | 커스텀 도메인 설정 (`www.sajukkum.com`) |
| `ads.txt` | 구글 애드센스 게시자 인증 |
| `sitemap.xml`, `robots.txt` | 검색엔진 최적화(SEO) |

## ✏️ 콘텐츠 수정 방법

- **꿈해몽 추가**: `saju.js`의 `DREAM_DB` 배열에 `{ k:[키워드], t:'제목', m:'풀이', tag:'분류' }` 추가
- **운세 문구 수정**: `saju.js`의 `FORTUNE_TOTAL` / `FORTUNE_MONEY` / `FORTUNE_LOVE` / `FORTUNE_WORK` 배열 편집
- **디자인 변경**: `style.css` 상단 `:root` 색상 변수 조정

## 🧮 사주 계산 기준

- 일주(日柱): 율리우스적일(JDN) 기반, 앵커 `2000-01-01 = 戊子(무자)`
- 년주: 입춘(2/4) 기준 전환 / 월주: 절기 근사 경계 + 오호둔 / 시주: 오서둔
- 절기 경계 출생은 월주가 하루 차이 날 수 있습니다 (참고용).
